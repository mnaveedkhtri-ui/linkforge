FAQ

Straight answers

Is LinkForge free?

The software itself is free to run yourself. You'll need Node.js, and if you want the AI copilot and Content Studio, your own free Groq API key.

Does it pull backlink or competitor data automatically?

Not yet. That needs a backlink data provider connected, which isn't wired up in this build. The pipeline, publisher marketplace, drafting, and campaigns are fully working.

Where is my outreach data stored?

In your own database, on whatever server you run LinkForge on. Nothing is sent anywhere except to Groq's API when you use the AI features.

Can a team use it together?

It's built for one workspace or a small team sharing a single deployment right now.
